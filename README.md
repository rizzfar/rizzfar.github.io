## **📌 Deskripsi Proyek**  
Proyek **Anagram Finder** ini dikembangkan sebagai bagian dari tugas akhir mata kuliah **Analisis dan Strategi Algoritma**. Sistem ini menggunakan pendekatan **Brute Force** untuk mencari anagram dari kata yang dimasukkan oleh pengguna dengan membandingkan hasil permutasi dengan dataset kamus yang telah disediakan.

## **🎯 Tujuan Proyek**  
- Mengimplementasikan algoritma **Brute Force** dalam pencarian anagram.  
- Menganalisis performa pencarian string menggunakan pendekatan permutasi.  
- Mengembangkan aplikasi berbasis web yang interaktif dan mudah digunakan.  

---

## **⚙️ Teknologi yang Digunakan**  
- **HTML** → Untuk struktur halaman web.  
- **CSS** → Untuk desain antarmuka pengguna (UI).  
- **JavaScript** → Untuk logika pencarian anagram.  
- **JSON** → Sebagai penyimpanan data kata dalam kamus.
